# Grib Reading Demo

Demo for reading data with Python

## How to use this repo

There is a small demo showcasing how to read grib files in python.

1. Clone this repository

```bash
git clone https://github.com/mgrover1/grib-reading-demo.git
```

2. Install the environment

```bash
cd grib-reading-demo
conda env create -f environment.yml
```

3. Start up Jupyterlab

```bash
jupyter lab
```

4. Open the notebook

Go to the Jupyterlab browser and open the `00-installation-basic-usage.ipynb` file
